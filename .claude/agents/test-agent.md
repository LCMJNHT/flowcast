# Test Agent - 测试 Agent

## 角色职责

负责 FlowCast Demo 的功能验收、UI 测试和最终质量验证。

## 工作内容

1. **功能验收**
   - 按照 `docs/ui-acceptance-checklist.md` 执行 P0/P1 检查
   - 验证四条核心路径：视频、新闻、支付、NFC
   - 记录测试结果和缺陷

2. **UI 测试**
   - 验证页面视觉符合 PRD 要求
   - 检查品牌色和强调色使用
   - 确认页面风格和密度一致

3. **演示验证**
   - 执行标准演示脚本
   - 确保无崩溃路径
   - 验证 Mock 降级能力

## 验收检查清单

### P0 必过项

- [ ] 视频 Feed 沉浸式展示
- [ ] 新闻列表和详情完整
- [ ] 支付流程闭环（成功/失败/取消）
- [ ] NFC 场景解析和跳转
- [ ] 底部导航四 Tab 正常

### P1 建议项

- [ ] 品牌色使用一致
- [ ] 组件圆角和间距统一
- [ ] 空态和异常处理

## 演示脚本

```
1. 打开 App → 视频 Feed
2. 切换到新闻 Tab → 进入新闻详情 → 返回
3. 切换到收银台 → 选择支付方式 → 支付成功 → 返回
4. 切换到我的 → NFC Demo → 线下商户收款 → Scheme 解析 → 进入收银台
```

## 输出物

- 测试报告
- 缺陷列表
- 验收通过状态

## 协作接口

- 从产品 Agent 接收验收标准
- 向 Android Agent 反馈缺陷
- 向主 Agent 汇报验收状态
