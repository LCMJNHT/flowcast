# FlowCast v2.1 产品需求分析

**分析人**: Product Agent  
**日期**: 2026-05-14  
**版本**: v2.1.0

---

## 1. 功能需求详情

### 1.1 视频播放功能

**需求描述**:
在视频 Feed 页面添加视频播放器 UI，支持播放/暂停控制。

**功能点**:
- 点击视频卡进入播放页
- 播放器支持播放/暂停
- 进度条显示和拖拽
- 全屏切换

**验收标准 (P0)**:
- [ ] 点击视频卡可进入播放页
- [ ] 播放/暂停按钮正常工作
- [ ] 进度条显示当前播放位置
- [ ] 支持横屏全屏播放

**优先级**: P0

---

### 1.2 新闻频道筛选

**需求描述**:
在新闻列表页面顶部添加频道筛选器，支持按频道筛选新闻。

**功能点**:
- 频道列表：全部/AI/零售科技/NFC/支付
- 选中态高亮显示
- 点击频道自动刷新列表

**验收标准 (P0)**:
- [ ] 频道筛选器横向滚动
- [ ] 选中态清晰可辨
- [ ] 点击频道后列表刷新
- [ ] 默认选中"全部"频道

**优先级**: P0

---

### 1.3 支付状态轮询

**需求描述**:
支付结果页面添加自动刷新机制，轮询支付状态。

**功能点**:
- 每 3 秒轮询一次支付状态
- 状态变化时自动更新 UI
- 超时处理（30 秒无响应）

**验收标准 (P0)**:
- [ ] 支付结果页自动刷新状态
- [ ] 状态变化时有动画提示
- [ ] 超时展示"处理中"状态

**优先级**: P1

---

### 1.4 NFC 记录页面

**需求描述**:
在"我的"页面添加 NFC 记录入口，展示历史触碰记录。

**功能点**:
- NFC 记录列表
- 每条记录展示时间、场景类型、解析结果
- 点击记录可查看详情

**验收标准 (P0)**:
- [ ] NFC 记录列表展示
- [ ] 记录按时间倒序排列
- [ ] 点击记录可查看详情页

**优先级**: P1
